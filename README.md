# 📈 A股自选股智能分析系统 (CLI Pro)

基于 AI 大模型的 A股/港股/美股自选股智能分析系统。本系统专为投资者打造，通过命令行（CLI）提供深度技术面分析、实时财讯情报及 AI 决策建议。

## 🚀 核心亮点

-   **真·异步架构**：基于 `asyncio` + `httpx`，高并发分析多只股票无阻塞。
-   **顶级情报源**：集成 **财联社实时电报**、通用搜索引擎（SerpAPI/Tavily）交叉验证。
-   **深度筹码识别**：自动识别单峰密集、超跌密集等专业筹码结构。
-   **多模型路由**：内置 LiteLLM 路由，支持 Gemini、DeepSeek、Claude 多 Key 负载均衡与自动故障切换。
-   **结构化配置**：使用 YAML 管理股票列表、模型路由及分析策略。

## 📦 运行模式

当前版本专注于 **高性能命令行 (CLI) 模式** 与 **API 服务模式**：
1.  **CLI 模式**：通过终端快速发起个股或批量分析，结果即时保存为 Markdown。
2.  **通知模式**：支持分析结果自动推送至飞书、企微、Telegram 等 10+ 渠道。
3.  **API 模式**：提供标准异步接口，方便集成至其他量化工作流。

> **注意**：本项目已全面重构为 CLI 优先架构，不再包含 Web 前端或桌面应用代码。

## 🛠️ 快速开始

### 1. 安装环境
```bash
pip install -r requirements.txt
```

### 2. 结构化配置
系统现在使用模块化 YAML 进行管理：
- `stocks.yaml`: 配置您的自选股列表。
- `settings.yaml`: 调整分析模式、并发限制及通知偏好。
- `litellm_config.yaml`: 配置您的 AI 模型与 API Key。
- `.env`: 存放最敏感的私密 Token。

### 3. 执行分析
```bash
python main.py --code 600519
```

## 📂 项目结构
- `src/core/`: 分析流水线核心。
- `src/analyzer/`: AI 逻辑与 Prompt 构建。
- `src/search/`: 异步搜索引擎集成。
- `data_provider/`: 多数据源适配器。
- `src/notification/`: 模块化通知推送系统。
